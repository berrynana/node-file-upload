# node-file-upload

## Purpose
The upload server supported multiple files, async.
### using nodejs
#### modules
- express
- formidable
- fs
- express-async-wrap
- path
- cors
- morgan
- http-errors
- cookie-parser

### cmd
1. `npm install`
2. `npm start`
3. http://localhost:3000/upload
