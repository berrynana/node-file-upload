# node-file-upload

## Purpose
The upload server supported multiple files, async.
using nodejs
modules
- express
- formidable
- fs
- express-async-wrap
- path
- cors
- morgan
- http-errors
- cookie-parser

* cmd
* `npm install`
* `npm start`
* http://localhost:3000/upload
